本文件夹存放了来自天大的关于Trimule200的原始文件，以及从中提取的Trimule200机器人的运动学逆解，速度逆解，加速度逆解，和动力学方程。从2023年10月开始锐意整理中~

【Trimule200_dynamics_original】来自天大的Trimule200机器人源文件，里面超多东西，咱们程序的主干就是从这个文件中提取出来的

【TriMule200】用于描述TriMule200机器人的类，可用于实例化TriMule200机器人对象，其中包含如下构造方法和类方法
*【TriMule200】用于建立机器人类的构造方法
*【T200InverseKinematics】用于求解机器人逆运动学问题的类方法，对应的单独的函数是【T200IK】，通过注释确认过单位
*【T200InverseVelocity】用于对机器人速度问题进行逆解的类方法，通过注释和绘图标记确认过单位
*【T200InverseAcceleration】用于对机器人加速度问题进行逆解的类方法，通过注释和绘图标记确认过单位
*【T200Dynamics】生成机器人刚体动力学方程，根据主动关节加速度求解主动关节驱动力的类方法，通过注释和绘图标记确认过单位

【TestforTriMule200】TriMule200机器人类的test程序，用于测试各种类方法是否可以正常运行
